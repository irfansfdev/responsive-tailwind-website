# 🚀 Creativo – Tailwind CSS Landing Page

A modern **responsive landing page** built using **HTML and Tailwind CSS**.  
This project demonstrates a complete SaaS / digital agency website layout with multiple UI components, responsive design, and interactive elements.

The goal of this project is to practice **modern frontend design using utility-first CSS with Tailwind** and create a professional multi-section landing page.

---

## ✨ Features

- Fully responsive design
- Modern SaaS / agency style layout
- Built using Tailwind CSS utility classes
- Clean component-based sections
- Interactive mobile navigation
- Hover animations and transitions
- Pricing cards
- Team section
- Testimonials
- Contact form UI
- FAQ section
- Newsletter subscription
- Professional footer

---

## 🧩 Sections Included

This landing page includes **15 sections**:

1. Navigation Bar (Responsive Mobile Menu)
